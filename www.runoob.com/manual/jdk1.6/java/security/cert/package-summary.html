软件包 java.security.cert

提供用于解析和管理证书、证书撤消列表 (CRL) 和证书路径的类和接口。

请参见：
          描述

接口摘要
CertPathBuilderResult 证书路径生成器算法的结果的规范。
CertPathParameters 证书路径算法参数的规范。
CertPathValidatorResult 证书路径验证器算法的结果的规范。
CertSelector 一个定义了一套选择 Certificate 的标准的选择器。
CertStoreParameters CertStore 参数的规范。
CRLSelector 一个定义了一套选择 CRL 的标准的选择器。
PolicyNode PKIX 证书路径验证算法所定义的、不可变的有效策略树节点。
X509Extension X.509 扩展的接口。
 

类摘要
Certificate 管理各种身份证书的抽象类。
Certificate.CertificateRep 用于序列化的替换 Certificate 类。
CertificateFactory 此类定义了用于从相关的编码中生成证书、证书路径 (CertPath) 和证书撤消列表 (CRL) 对象的 CertificateFactory 功能。
CertificateFactorySpi 此类定义了 CertificateFactory 类的服务提供者接口 (SPI)。
CertPath 不可变的证书序列（证书路径）。
CertPath.CertPathRep 用于序列化的替换 CertPath 类。
CertPathBuilder 构建证书路径（也称为证书链）的类。
CertPathBuilderSpi CertPathBuilder 类的服务提供者接口 (SPI)。
CertPathValidator 验证证书路径（也称为证书链）的类。
CertPathValidatorSpi CertPathValidator 类的服务提供者接口 (SPI)。
CertStore 用于从存储库中获取 Certificate 和 CRL 的类。
CertStoreSpi CertStore 类的服务提供者接口 (SPI)。
CollectionCertStoreParameters 用作 Collection CertStore 算法输入的参数。
CRL 此类是具有不同格式但很常用的证书撤消列表 (CRL) 的抽象。
LDAPCertStoreParameters 用作 LDAP CertStore 算法的输入的参数。
PKIXBuilderParameters 用作 PKIX CertPathBuilder 算法的输入的参数。
PKIXCertPathBuilderResult 此类表示 PKIX 证书路径生成器算法执行成功时所返回的结果。
PKIXCertPathChecker 对 X509Certificate 执行一种或多种检查的抽象类。
PKIXCertPathValidatorResult 此类表示 PKIX 证书路径验证算法执行成功时所返回的结果。
PKIXParameters 用作 PKIX CertPathValidator 算法的输入的参数。
PolicyQualifierInfo 由 ASN.1 PolicyQualifierInfo 结构所表示的、不可变的策略限定符。
TrustAnchor 一个信任定位点或最受信任的证书颁发机构 (CA)。
X509Certificate X.509 证书的抽象类。
X509CertSelector 选择与所有指定标准匹配的 X509Certificates 的 CertSelector。
X509CRL X.509 证书撤消列表 (CRL) 的抽象类。
X509CRLEntry 用于 CRL（证书撤消列表）中已撤消证书的抽象类。
X509CRLSelector 选择与所有指定标准相匹配的 X509CRL 的 CRLSelector。
 

异常摘要
CertificateEncodingException 证书编码异常。
CertificateException 此异常指示各种证书问题中的一种。
CertificateExpiredException 证书过期异常。
CertificateNotYetValidException 证书尚无效异常。
CertificateParsingException 证书解析异常。
CertPathBuilderException 指示在使用 CertPathBuilder 构建证书路径时遇到了某种问题的异常。
CertPathValidatorException 指示在验证一个证书路径时遇到了某种问题的异常。
CertStoreException 指示从 CertStore 中获取证书和 CRL 时遇到了某种问题的异常。
CRLException CRL（证书撤消列表）异常。
 

软件包 java.security.cert 的描述

提供用于解析和管理证书、证书撤消列表 (CRL) 和证书路径的类和接口。它包含对 X.509 v3 证书和 X.509 v2 CRL 的支持。

包规范

  • 加密体系结构
  • RFC 2459:X.509 证书和 CRL 配置文件

相关文档

有关 X.509 证书和 CRL 的信息，请参阅：
  • http://www.ietf.org/rfc/rfc2459.txt
  • X.509 Certificates and CRLs
