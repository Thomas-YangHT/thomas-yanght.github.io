抽象类 Number 是 BigDecimal、BigInteger、Byte、Double、Float、Integer、Long 和 Short 类的超类。

Number 的子类必须提供将表示的数值转换为 byte、double、float、int、long 和 short 的方法。

从以下版本开始：
JDK1.0
