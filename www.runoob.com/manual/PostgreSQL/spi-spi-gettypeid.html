SPI_gettypeid

名称

SPI_gettypeid -- 返回声明字段的数据类型OID

大纲

Oid SPI_gettypeid(TupleDesc rowdesc, int colnumber)

描述

SPI_gettypeid返回声明字段的数据类型的OID。

参数

TupleDesc rowdesc

输入行描述

int colnumber

字段编号(从 1 开始记)

返回值

声明字段的数据类型的OID，或者错误时是InvalidOid。 在出错的时候，SPI_result设置为SPI_ERROR_NOATTRIBUTE。
