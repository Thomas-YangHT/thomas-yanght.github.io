java.io
接口 ObjectOutput

所有超级接口：
DataOutput
所有已知实现类：
ObjectOutputStream
public interface ObjectOutput
extends DataOutput

ObjectOutput 扩展 DataOutput 接口以包含对象的写入操作。DataOutput 包括基本类型的输出方法；ObjectOutput 扩展了该接口，以包含对象、数组和 String 的输出方法。

从以下版本开始：
JDK1.1
另请参见：
InputStream, ObjectOutputStream, ObjectInputStream
