软件包 java.awt.print

为通用的打印 API 提供类和接口。

请参见：
          描述

接口摘要
Pageable Pageable 实现表示要打印的页面集合。
Printable Printable 接口由当前页面 painter 的 print 方法实现，该方法由打印系统调用，以呈现页面。
PrinterGraphics PrinterGraphics 接口由被传递到 Printable 对象以呈现页面的 Graphics 对象实现。
 

类摘要
Book Book 类提供文档的表示形式，该文档的页面可以使用不同的页面格式和页面 painter。
PageFormat PageFormat 类描述要打印的页面大小和方向。
Paper Paper 类描述一张纸的物理特征。
PrinterJob PrinterJob 类是控制打印的主要类。
 

异常摘要
PrinterAbortException PrinterAbortException 类是 PrinterException 的子类，用于指示用户或应用程序在打印过程中终止打印作业。
PrinterException PrinterException 类及其子类用于指示打印系统中发生的异常情况。
PrinterIOException PrinterIOException 类是 PrinterException 的子类，用于指示在打印过程中发生了某种 IO 错误。
 

软件包 java.awt.print 的描述

为通用的打印 API 提供类和接口。该 API 包括以下功能：

  • 指定文档类型的功能
  • 控制页面设置和页面格式的机制
  • 管理作业控制对话的功能
