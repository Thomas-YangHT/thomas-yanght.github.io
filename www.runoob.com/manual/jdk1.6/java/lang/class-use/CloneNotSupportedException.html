类 java.lang.CloneNotSupportedException
的使用

使用 CloneNotSupportedException 的软件包
java.awt.datatransfer 提供在应用程序之间和在应用程序内部传输数据的接口和类。 
java.lang 提供利用 Java 编程语言进行程序设计的基础类。 
java.rmi.server 提供支持服务器端 RMI 的类和接口。 
java.security 为安全框架提供类和接口。 
java.util 包含 collection 框架、遗留的 collection 类、事件模型、日期和时间设施、国际化和各种实用工具类（字符串标记生成器、随机数生成器和位数组）。 
javax.crypto 为加密操作提供类和接口。 
javax.swing 提供一组“轻量级”（全部是 Java 语言）组件，尽量让这些组件在所有平台上的工作方式都相同。 
javax.swing.text 提供类 HTMLEditorKit 和创建 HTML 文本编辑器的支持类。 
javax.swing.tree 提供处理 javax.swing.JTree 的类和接口。 
 

java.awt.datatransfer 中 CloneNotSupportedException 的使用
 

抛出 CloneNotSupportedException 的 java.awt.datatransfer 中的方法
 Object DataFlavor.clone()
          返回此 DataFlavor 的副本。
 

java.lang 中 CloneNotSupportedException 的使用
 

抛出 CloneNotSupportedException 的 java.lang 中的方法
protected  Object Enum.clone()
          抛出 CloneNotSupportedException。
protected  Object Object.clone()
          创建并返回此对象的一个副本。
 

java.rmi.server 中 CloneNotSupportedException 的使用
 

java.rmi.server 中 CloneNotSupportedException 的子类
 class ServerCloneException
          如果在复制 UnicastRemoteObject 的过程中发生远程异常，则抛出 ServerCloneException。
 

抛出 CloneNotSupportedException 的 java.rmi.server 中的方法
 Object UnicastRemoteObject.clone()
