DROP TEXT SEARCH PARSER

名称

DROP TEXT SEARCH PARSER -- 删除一个文本搜索解析器

大纲

DROP TEXT SEARCH PARSER [ IF EXISTS ] name [ CASCADE | RESTRICT ]

描述

DROP TEXT SEARCH PARSER删除一个现有文本搜索解析器。你必须是超级用户才能使用此命令。

参数

IF EXISTS

如果指定的文本搜索解析器不存在，那么发出一个notice而不是抛出一个错误。

name

一个现有文本搜索解析器的名称（可有模式修饰）。

CASCADE

级联删除依赖该文本搜索解析器的对象。

RESTRICT

若有任何对象依赖文本搜索解析器则拒绝删除它。这是默认情况。

例子

删除文本搜索解析器my_parser: 

DROP TEXT SEARCH PARSER my_parser;

若有使用该解析器的任何现有文本搜索配置，该命令将不会成功。 添加CASCADE来删除配置连同解析器。

兼容性

在SQL标准中没有DROP TEXT SEARCH PARSER语句。

又见

ALTER TEXT SEARCH PARSER, CREATE TEXT SEARCH PARSER
