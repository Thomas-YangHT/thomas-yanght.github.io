类 MethodDescriptor

java.lang.Object
  java.beans.FeatureDescriptor

public class MethodDescriptor
extends FeatureDescriptor

MethodDescriptor 描述了一种特殊方法，即 Java Bean 支持从其他组件对其进行外部访问。
