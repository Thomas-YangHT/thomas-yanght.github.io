类 java.awt.dnd.DragSourceEvent
的使用

使用 DragSourceEvent 的软件包
java.awt.dnd Drag 和 Drop 是一种直接操作动作，在许多图形用户界面系统中都会遇到它，它提供了一种机制，能够在两个与 GUI 中显示元素逻辑相关的实体之间传输信息。 
 

java.awt.dnd 中 DragSourceEvent 的使用
 

java.awt.dnd 中 DragSourceEvent 的子类
 class DragSourceDragEvent
          通过 DragSourceContext，将 DragSourceDragEvent 从 DragSourceContextPeer 传递给向 DragSourceContext 及其关联 DragSource 注册的 DragSourceListener。
 class DragSourceDropEvent
          通过 DragSourceContext，将 DragSourceDropEvent 从 DragSourceContextPeer 传递给向 DragSourceContext 及其关联 DragSource 注册的 DragSourceListener 的 dragDropEnd 方法。
 

参数类型为 DragSourceEvent 的 java.awt.dnd 中的方法
 void DragSourceAdapter.dragExit(DragSourceEvent dse)
          光标的热点离开放置点（与平台相关）时调用该方法。
 void DragSourceContext.dragExit(DragSourceEvent dse)
          对向此 DragSourceContext 及其关联 DragSource 注册的 DragSourceListener 调用 dragExit，并将指定的 DragSourceEvent 传递给它们。
 void DragSourceListener.dragExit(DragSourceEvent dse)
          光标的热点离开放置点（与平台相关）时调用该方法。
