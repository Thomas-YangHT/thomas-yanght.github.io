addAttribute

void addAttribute(Object name,
                  Object value)
创建一个与此属性集类似的新属性集，不同之处在于它包含具有给定名称和值的属性。该对象必须是不可变的，或者说不能由任何客户端改变。

参数：
name - 名称
value - 值
