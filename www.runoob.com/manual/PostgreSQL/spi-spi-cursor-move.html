SPI_cursor_move

名称

SPI_cursor_move -- 移动一个游标

大纲

void SPI_cursor_move(Portal portal, bool forward, long count)

描述

SPI_cursor_move忽略游标中的一些行。 这个函数等效于 SQL 命令MOVE的一个子集（参阅 SPI_scroll_cursor_move获取更多功能性）。

参数

Portal portal

包含游标的入口

bool forward

向前移动时为真，向后移动时为假

long count

移动的最大行数

注意

如果游标的规划没有带有CURSOR_OPT_SCROLL选项创建， 那么向后移动会失败。
