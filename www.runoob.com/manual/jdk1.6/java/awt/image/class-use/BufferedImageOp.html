接口 java.awt.image.BufferedImageOp
的使用

使用 BufferedImageOp 的软件包
java.awt 包含用于创建用户界面和绘制图形图像的所有类。 
java.awt.image 提供创建和修改图像的各种类。 
 

java.awt 中 BufferedImageOp 的使用
 

参数类型为 BufferedImageOp 的 java.awt 中的方法
abstract  void Graphics2D.drawImage(BufferedImage img, BufferedImageOp op, int x, int y)
          呈现使用 BufferedImageOp 过滤的 BufferedImage。
 

java.awt.image 中 BufferedImageOp 的使用
 

实现 BufferedImageOp 的 java.awt.image 中的类
 class AffineTransformOp
          此类使用仿射转换来执行从源图像或 Raster 中 2D 坐标到目标图像或 Raster 中 2D 坐标的线性映射。
 class ColorConvertOp
          此类对源图像中的数据执行逐像素的颜色转换。
 class ConvolveOp
          此类实现从源到目标的卷积。
 class LookupOp
          此类实现从源到目标的查找操作。
 class RescaleOp
          通过将每个像素的样本值乘以一个缩放因子，然后加上一个偏移量，此类对源图像中数据进行逐像素重缩放。
 

返回 BufferedImageOp 的 java.awt.image 中的方法
 BufferedImageOp BufferedImageFilter.getBufferedImageOp()
          返回 BufferedImageOp。
 

参数类型为 BufferedImageOp 的 java.awt.image 中的构造方法
BufferedImageFilter(BufferedImageOp op)
          构造具有指定单一源图像/单一目标图像操作符的 BufferedImageFilter。
