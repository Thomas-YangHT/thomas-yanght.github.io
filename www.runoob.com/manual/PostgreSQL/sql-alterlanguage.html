ALTER LANGUAGE

名称

ALTER LANGUAGE -- 修改一个过程语言的定义

大纲

ALTER [ PROCEDURAL ] LANGUAGE name RENAME TO new_name
ALTER [ PROCEDURAL ] LANGUAGE name OWNER TO new_owner

描述

ALTER LANGUAGE修改一个语言的定义。 目前唯一的功能就是重命名语言或是设置新的所有者。 只有超级用户或者语言的所有者 可以使用ALTER LANGUAGE。

参数

name

语言的名字。

new_name

语言的新名字。

new_owner

语言的新的所有者。

兼容性

SQL标准里没有ALTER LANGUAGE语句。。

参见

CREATE LANGUAGE, DROP LANGUAGE
