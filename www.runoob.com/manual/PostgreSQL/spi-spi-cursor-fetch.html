SPI_cursor_fetch

名称

SPI_cursor_fetch -- 从一个游标里抓取一些行

大纲

void SPI_cursor_fetch(Portal portal, bool forward, long count)

描述

SPI_cursor_fetch从一个游标里抓取一些行。 这个函数等效于 SQL 命令FETCH的一个子集（参阅 SPI_scroll_cursor_fetch获取更多功能性）。

参数

Portal portal

包含游标的入口

bool forward

向前抓取时为真，向后抓取时为假

long count

要抓取的最大行数

返回值

成功时，像SPI_execute那样设置 SPI_processed和SPI_tuptable

注意

如果游标的规划没有带有CURSOR_OPT_SCROLL选项创建， 那么向后抓取会失败。
