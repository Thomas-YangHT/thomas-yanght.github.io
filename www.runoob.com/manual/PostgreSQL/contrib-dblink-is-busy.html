dblink_is_busy

名称

dblink_is_busy -- 检查是否连接忙于异步查询

大纲

dblink_is_busy(text connname) returns int

描述

dblink_is_busy测试是否异步查询在进行中。

参数

conname

检查连接的名字。

返回值

如果连接繁忙，则返回1，如果不繁忙，则返回0。 如果该函数返回0，那么保证dblink_get_result将不阻塞。

例子

SELECT dblink_is_busy('dtest1');
