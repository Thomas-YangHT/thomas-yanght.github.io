public interface TabExpander

允许不同类型的制表符扩展实现的简单接口。

方法摘要
 float nextTabStop(float x, int tabOffset)
          给定一个引用位置，返回下一个制表位 (tab stop) 位置。
 

方法详细信息

nextTabStop

float nextTabStop(float x,
                  int tabOffset)
给定一个引用位置，返回下一个制表位 (tab stop) 位置。值以点为单位表示。

参数：
x - 以点为单位的位置，该值 >= 0
tabOffset - 文本流中的位置，制表符在该处出现，该值 >= 0。
返回：
下一个制表位，该值 >= 0
